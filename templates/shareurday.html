ShareUrDay

What is ShareUrDay?

ShareUrDay is an Android application that lets the user create a diary of their own and share it with people from around the world, anonymously. The application lets the user create diary messages and share it with other users who can rate and reply the message. The user can also retrieve a diary message from other users by random, in the case of random user can enjoy reading new diary messages all the time. All diary messages written by the user is stored privately for themselves, so they can one day read them again. This application would be beneficial to users in many ways such as socialising, learning about new cultures (reading other's diary messages) and would form a better understanding of people from different areas. ShareUrDay is simple, effective and easy to use.

Why ShareUrDay?

In the 19th century, we did not have any particular way to conserve the past without the use of paper. The ongoing war has silenced many for over decades. People did not speak out, so they wrote them on a piece of paper. People were using diaries to preserve their past life. Since then diaries were becoming less common and eventually forgotten. Due to the increase of mobile applications,ShareUrDay was inspired to conserve a user's personal life journey from young to old and allows them to share their feelings while socialising at the same time and being anonymous to others who uses ShareUrDay.

Anonymity

ShareUrDay does not share any personal information, users has a choice to keep their private information on their user profile, but their user profile is not shared with anyone. The only information that will identify the user are:

How do I Install?

Option 1: Mobile installation method (approx. 3mins)

  1. Enable Your mobile to allow Unknown source in setting -> security -> Unknown source (tick).
  2. Go to www.shareurday.heroku.com and download the ShareUrDay.apk file.
  3. Once download is complete, click on the file and install it.

Option 2: Web installation method (approx. 10-15mins)

  1. Connect your device by usb to laptop.
  2. Create a new folder inside your device using your laptop.
  3. Drag the APK file to the folder you created.
  4. Unplug the usb.
  5. Go to setting->security then tick the "Allow Unknown Source" option.
  6. Now find the folder you put your apk in by using a folder directory application on your device.
  7. Click on the .apk file and accept the conditions. Now the app is created in your device.
